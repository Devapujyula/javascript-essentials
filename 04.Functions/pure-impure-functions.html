Pure and Impure Functions

Pure Function : A pure function is a function that:
  1. Always returns the same output for the same input.
  2. Does not cause side effects (like modifying global variables, DOM, files, DB, etc.).
A pure function is predictable and doesn't depend on or modify any outside state. It's easier to test and debug.

An impure function depends on or changes external state. This makes the output unpredictable, and harder to test or reuse.
